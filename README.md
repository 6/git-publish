git-publish
=============

> `git publish`

Export docs to gh-pages.

Getting Started
----------

#### Install
```bash
git clone this && cd this && make install
git publish init # create .gitpublish
```

####Use
```bash
git publish
```

Options
-----------

```bash
BUILD_COMMAND=
EXPORT_DIR=
REMOTE_REF='origin'
REMOTE_BRANCH='gh-pages'
```
